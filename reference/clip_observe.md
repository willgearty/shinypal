# Add an observer to a copy button

Generates an event observer that watches for when a copy button is
clicked. Upon clicking, the `code_expr` is evaluated and copied to the
clipboard using
[`clipr::write_clip()`](http://matthewlincoln.net/clipr/reference/write_clip.md).

## Usage

``` r
clip_observe(ind, code_expr)
```

## Arguments

- ind:

  The index of the step.

- code_expr:

  An expression (generated by
  [`rlang::expr()`](https://rlang.r-lib.org/reference/expr.html) that,
  when evaluated, returns content to be copied (such as the output from
  [`shinymeta::expandChain()`](https://rstudio.github.io/shinymeta/reference/expandChain.html).

## Value

Called for its side effects; invisibly returns the observer.

``` r

```
